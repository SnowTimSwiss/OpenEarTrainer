# Open Ear Trainer

Kleine lokale Web-App zum Üben von:

- Intervalle
- Dreiklänge mit Umkehrungen
- Vierklänge mit Umkehrungen
- Tonmaterial / Modi / Skalen

## Starten

`index.html` direkt im Browser öffnen.

## Funktionen

- Modus `Hören`: Aufgabe wird abgespielt, du wählst die richtige Antwort.
- Modus `Singen`: Grundton wird abgespielt, die App sagt dir was du singen sollst, danach kannst du die Lösung anhören.
- Zufälliges Shuffle bei jeder neuen Aufgabe.
- Statistik pro Thema und pro Akkord / Intervall / Skala.
- Statistik wird lokal im Browser gespeichert.

## Hinweis

Die erste Version bewertet Gesang noch nicht automatisch per Mikrofon. Sie ist absichtlich robust und einfach gehalten: hören, singen, Lösung anhören, weitermachen.
